Shpee Bot Commands

Fun

.8ball [question]
Ask a yes/no question, get an answer

.level
Well, let's see how many levels you have!

.points
Well, let's see how many points you have!

Miscelaneous

.embed [description]
Embeds something

.stats
Gives some useful bot statistics

.permlevel
Returns your permission level

.userinfo [@user]
Will give you stats about your (or another guild member's) Discord account

.say [anything]
It says what you say

.tags [tag name]
Allows you to add tags (only single-word tags allowed), then run [prefix]tags [tag name] to display the tag, delete the tags if needed, and display more info on the tags

.serverinfo
Will give you info about this server

.ping
For checking response time

.test
Testing, testing

Utilities

.google [search]
Too lazy to leave Discord to Google something?

.expand [shortened url]
Get the long URL that a short URL redirects to

.urban [term]
Gets a definition from UrbanDictionary

.shorten [URL] [title]
Shortens a given URL using is.gd

.remindme [s/m/h/d] [reminder]
For when you need to remember something

.weather [location]
Get the weather for a location
